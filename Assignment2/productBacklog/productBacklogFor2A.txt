. As a user, I can view shell feedback so that I can understand my interactions with the shell

. As a user, I can input characters into the shell so that I can interact with the shell 

. As a user, I can exit the shell by calling exit command so that exiting the shell is simple

. As a user, I can create a new directory relative to the current working directory by calling mkdir command without specifying a full path

. As a user, I can create a new directory by calling mkdir command with a full path specified 

. As a user, I can change my current working directory by writing out a full path

. As a user, I can change my current working directory by writing a relative path

. As a user, I can print out the contents of a directory

. As a user, I can print out the complete path of my current working directory

. As a user, I can save the current working directory to a directory stack

. As a user, I can extract directories from the directory stack

. As a user, I can print out the history of all my inputs

. As a user, I can print out the contents of files

. As a user, I can print any string onto the shell

. As a user, I can create a new text file containing any string I input

. As a user, I can erase the contents of text files

. As a user, I can overwrite the contents of any text file with any string I input

. As a user, I can print the documentation of all commands by calling the man command so that I can understand how to properly interact with the shell
